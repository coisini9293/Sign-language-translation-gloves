## 3. 网站设计风格

### 3.1 配色方案
主色调：深邃黑色，象征科技的专业与可靠
辅助色：纯净白色，营造简洁现代的视觉感受

### 3.2 视觉细节
圆角设计：适度圆角增加亲和力，保持专业感
阴影层次：清晰的阴影效果增强空间层次
图标风格：线性图标配合填充图标，统一视觉语言

### 3.3 整体布局
首页采用全屏沉浸式设计，其他页面使用卡片式布局
注重视觉冲击力与信息层次的平衡
