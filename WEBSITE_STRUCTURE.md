# 手语翻译手套项目官方网站结构

## 网站导航

网站现在包含5个主要页面，导航栏横幅包括：

1. **首页** (`/`)
2. **产品介绍** (`/product`)
3. **技术原理** (`/technology`)
4. **背景故事** (`/story`)
5. **联系我们** (`/contact`)

## 页面详情

### 1. 首页 (Home)
- **路径**: `/`
- **内容**:
  - 大标题："手语翻译手套 - 让每一次沉默的比划，都被世界听见"
  - 产品简介
  - 核心价值展示（准确率>92%、成本<$50、实时翻译）
  - 产品工作流程视频展示
  - 行动号召

### 2. 产品介绍 (Product)
- **路径**: `/product`
- **内容**:
  - **技术验证**: 识别准确率、响应速度、用户满意度数据
  - **用户反馈**: 特教老师和听障用户的真实评价，测试现场照片

    - 听障儿童的课堂提问
    - 聋哑人士的医院就医
    - 听障员工的职场沟通

  - **合作邀请**: 寻求学术机构、特教学校、企业赞助商合作

### 3. 技术原理 (Technology)
- **路径**: `/technology`
- **内容**:

    - 弯曲传感器详解及演示图
    - 陀螺仪详解及演示图
  - 处理中心：ESP32主控板介绍
  - 输出模块：语音合成模块
  - 算法原理：数据处理流程

### 4. 背景故事 (Story)
- **路径**: `/story`
- **内容**:

  - **我们的使命**: 构建无障碍沟通环境的愿景
  - **核心团队**: 
    - Carolin - 商业负责人
    - Stephen - 技术负责人
    - 王芳 - 产品设计师

### 5. 联系我们 (Contact)
- **路径**: `/contact`
- **内容**:
  - 联系表单（姓名、邮箱、主题、消息）
  - 联系方式（邮箱、社交媒体）
  - 常见问题解答

## 设计风格

### 配色方案
- **主色调**: 蓝色 (HSL: 217 91% 60%) - 科技感和专业性
- **背景色**: 浅蓝灰色 (HSL: 214 32% 91%) - 柔和舒适
- **辅助色**: 深色文字 (HSL: 222 47% 11%) - 清晰易读

### 视觉特点
- 半透明导航栏，带有毛玻璃效果
- 圆角设计（0.75rem）增加亲和力
- 柔和的阴影效果增强层次感
- 响应式布局，适配桌面和移动设备
- 悬停效果和过渡动画提升交互体验

## 技术栈

- **前端框架**: React + TypeScript
- **样式**: Tailwind CSS
- **UI组件**: shadcn/ui
- **路由**: React Router
- **后端**: Supabase (数据库和API)
- **构建工具**: Vite

## 数据库表

1. **blog_posts** - 博客文章（已创建但未在当前导航中使用）
2. **user_cases** - 用户案例
3. **resources** - 资源文件
4. **contact_submissions** - 联系表单提交
